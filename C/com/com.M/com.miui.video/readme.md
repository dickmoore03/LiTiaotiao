## 基础规则

快速复制:
```
{"popup_rules":
    [
        {"id":"青少年模式","action":"我知道了"},        
        {"id":"tv_ad_label","action":"iv_close"},
        {"id":"打开&借钱","action":"iv_close"},
        {"id":"抢疯&淘宝","action":"iv_close"},
        {"id":"特价&不停","action":"iv_close"}
    ]
}
```

### {"id":"青少年模式","action":"我知道了"}

### {"id":"tv_ad_label","action":"iv_close"}  

### {"id":"打开&借钱","action":"iv_close"}

### {"id":"抢疯&淘宝","action":"iv_close"}
关闭“淘宝”横幅广告


### {"id":"特价&不停","action":"iv_close"}
